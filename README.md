# Unit Zero

Unit Zero is an experimental retro-inspired music player built with HTML, CSS and JavaScript.

## Features

- Custom music player UI
- Interactive controls
- Playlist system (work in progress)
- Audio playback
- Retro/cyber-inspired visual design

## Goals

This project was created for:
- Study and practice
- UI/UX experimentation
- JavaScript learning
- Future expansion into a more complete multimedia player

## Technologies

- HTML5
- CSS3
- JavaScript

## Status

Early development (v0.01)
